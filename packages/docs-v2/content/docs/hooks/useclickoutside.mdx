## Installation

```bash
npx useverse@latest add useClickOutside
```

## Usage

 
  ```tsx 
    useClickOutside(ref, () => console.log('click outside')); 
  ``` 
   
  ```tsx 
    const { ref } = useClickOutside<HTMLDivElement>(() => console.log('click outside')); 
  ```
