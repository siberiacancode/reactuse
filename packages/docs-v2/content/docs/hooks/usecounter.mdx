## Installation

```bash
npx useverse@latest add useCounter
```

## Usage

 
  ```tsx 
    const { count, dec, inc, reset, set } = useCounter(5); 
  ``` 
   
  ```tsx 
    const { count, dec, inc, reset, set } = useCounter({ initialValue: 5, min: 0, max: 10 }); 
  ```
