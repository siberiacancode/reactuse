## Installation

```bash
npx useverse@latest add useCssVar
```

## Usage

 
  ```tsx 
    const { ref, value, set, remove } = useCssVar('--color', 'red'); 
  ``` 
   
  ```tsx 
    const { value, set, remove } = useCssVar(ref, '--color', 'red'); 
  ```
