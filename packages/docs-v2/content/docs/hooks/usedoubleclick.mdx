## Installation

```bash
npx useverse@latest add useDoubleClick
```

## Usage

 
  ```tsx 
    useDoubleClick(ref, () => console.log('double clicked')); 
  ``` 
   
  ```tsx 
    const { ref } = useDoubleClick(() => console.log('double clicked')); 
  ```
