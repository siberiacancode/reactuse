## Installation

```bash
npx useverse@latest add useFocusTrap
```

## Usage

 
  ```tsx 
    const { active, disable, toggle, enable } = useFocusTrap(ref, true); 
  ``` 
   
  ```tsx 
    const { ref, active, disable, toggle, enable } = useFocusTrap(true); 
  ```
