## Installation

```bash
npx useverse@latest add useFullscreen
```

## Usage

 
  ```tsx 
    const { enter, exit, toggle, value } = useFullscreen(ref); 
  ``` 
   
  ```tsx 
    const { ref, enter, exit, toggle, value } = useFullscreen(); 
  ```
