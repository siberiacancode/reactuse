## Installation

```bash
npx useverse@latest add useHash
```

## Usage

 
  ```tsx 
    const { value, set } = useHash("initial"); 
  ``` 
   
  ```tsx 
    const { value, set } = useHash("initial", (newHash) => console.log('callback')); 
  ``` 
   
  ```tsx 
    const { value, set } = useHash(); 
  ``` 
   
  ```tsx 
    const { value, set } = useHash((newHash) => console.log('callback')); 
  ```
