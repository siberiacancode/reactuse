## Installation

```bash
npx useverse@latest add useHover
```

## Usage

 
  ```tsx 
    const hovering = useHover(ref, () => console.log('callback')); 
  ``` 
   
  ```tsx 
    const hovering = useHover(ref, options); 
  ``` 
   
  ```tsx 
    const { ref, value } = useHover(() => console.log('callback')); 
  ``` 
   
  ```tsx 
    const { ref, value } = useHover(options); 
  ```
