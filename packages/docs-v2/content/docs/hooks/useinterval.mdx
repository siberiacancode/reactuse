## Installation

```bash
npx useverse@latest add useInterval
```

## Usage

 
  ```tsx 
    const { active, pause, resume, toggle } = useInterval(() => console.log('inside interval'), 2500); 
  ``` 
   
  ```tsx 
    const { active, pause, resume, toggle } = useInterval(() => console.log('inside interval'), { interval: 2500 }); 
  ```
