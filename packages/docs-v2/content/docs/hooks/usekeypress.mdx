## Installation

```bash
npx useverse@latest add useKeyPress
```

## Usage

 
  ```tsx 
    const isKeyPressed = useKeyPress(ref, 'a'); 
  ``` 
   
  ```tsx 
    const { pressed, ref } = useKeyPress('a'); 
  ```
