## Installation

```bash
npx useverse@latest add useKeyPressEvent
```

## Usage

 ```tsx 
 useKeyPressEvent(ref, 'Enter', () => console.log('pressed')); 
 ``` 
 ```tsx 
 const ref = useKeyPressEvent('Enter', (event) => console.log('pressed')); 
 ```
