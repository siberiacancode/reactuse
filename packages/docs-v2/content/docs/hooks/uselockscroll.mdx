## Installation

```bash
npx useverse@latest add useLockScroll
```

## Usage

 
  ```tsx 
    const { lock, unlock, value, toggle } = useLockScroll(ref); 
  ``` 
   
  ```tsx 
    const { ref, lock, unlock, value, toggle } = useLockScroll(); 
  ```
