## Installation

```bash
npx useverse@latest add useMeasure
```

## Usage

 
  ```tsx 
    const { x, y, width, height, top, left, bottom, right } = useMeasure(ref); 
  ``` 
   
  ```tsx 
    const { ref, x, y, width, height, top, left, bottom, right } = useMeasure(); 
  ```
