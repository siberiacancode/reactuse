## Installation

```bash
npx useverse@latest add useOptimistic
```

## Usage

 
  ```tsx 
    const [optimisticValue, updateOptimistic] = useOptimistic<number>(count, (currentState, optimisticValue) => currentState + optimisticValue); 
  ```
