## Installation

```bash
npx useverse@latest add useOtpCredential
```

## Usage

 
  ```tsx 
    useOtpCredential((credential) => console.log(credential)); 
  ``` 
   
  ```tsx 
    useOtpCredential({ onSuccess: (credential) => console.log(credential), onError: (error) => console.log(error) }); 
  ```
