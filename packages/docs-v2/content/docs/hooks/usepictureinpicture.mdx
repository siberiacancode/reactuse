## Installation

```bash
npx useverse@latest add usePictureInPicture
```

## Usage

 
  ```tsx 
    const { open, supported, enter, exit, toggle } = usePictureInPicture(videoRef); 
  ``` 
   
  ```tsx 
    const { ref, open, supported, enter, exit, toggle } = usePictureInPicture(); 
  ```
