## Installation

```bash
npx useverse@latest add useRightClick
```

## Usage

 
  ```tsx 
    useRightClick(ref, () => console.log('clicked')); 
  ``` 
   
  ```tsx 
    const ref = useRightClick(() => console.log('clicked')); 
  ```
