## Installation

```bash
npx useverse@latest add useScrollIntoView
```

## Usage

 
  ```tsx 
    const { trigger } = useScrollIntoView(ref); 
  ``` 
   
  ```tsx 
    const { ref, trigger } = useScrollIntoView<HTMLDivElement>(); 
  ```
