## Installation

```bash
npx useverse@latest add useStopwatch
```

## Usage

 
  ```tsx 
    const { seconds, minutes, start, pause, reset } = useStopwatch(1000, { enabled: false }); 
  ``` 
   
  ```tsx 
    const { seconds, minutes, start, pause, reset } = useStopwatch({ initialTime: 1000, enabled: false }); 
  ```
