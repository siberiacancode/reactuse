## Installation

```bash
npx useverse@latest add useTextDirection
```

## Usage

 
  ```tsx 
    const { value, set, remove } = useTextDirection(ref); 
  ``` 
   
  ```tsx 
    const { ref, value, set, remove } = useTextDirection(); 
  ```
