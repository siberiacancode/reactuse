Reactuse provides a dedicated skill for coding agents so they can choose the right hook faster and follow consistent usage patterns.

## Reactuse skill

- Registry page: [reactuse skill](https://skills.sh/siberiacancode/agent-skills/reactuse)
- Source repository: [siberiacancode/agent-skills](https://github.com/siberiacancode/agent-skills)

## Install

```bash
npx skills add https://github.com/siberiacancode/agent-skills --skill reactuse
```

## What this skill gives you

- Hook-first decision model before writing custom logic
- Invocation priority guidance (`HIGH`, `MEDIUM`, `LOW`)
- Structured coverage by domains: `async`, `browser`, `elements`, `lifecycle`, `state`, `utilities`, `sensors`, `time`, `user`, `debug`, and `helpers`
- Direct mapping from requirement to a concrete Reactuse API
